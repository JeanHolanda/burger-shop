# 🍔 Burger Shop

## 🇺🇸 About the project

A simple burger ordering interface I built to practice JavaScript array methods in real scenarios, such as:

* `forEach`
* `map`
* `filter`
* `reduce`

The project simulates a menu where users can view products, filter options, and calculate totals dynamically.

---

## 🇧🇷 Sobre o projeto

Uma interface simples de pedidos de hamburguer, desenvolvida com **HTML, CSS e JavaScript**, com foco na prática de métodos de manipulação de arrays em cenários reais, como:

* `forEach`
* `map`
* `filter`
* `reduce`

O projeto simula um cardápio onde o usuário pode visualizar produtos, filtrar opções e calcular valores dinamicamente.

---

## 🎯 Purpose

This project was created to strengthen my JavaScript fundamentals, especially working with array methods and DOM manipulation.

---

## 🚀 Technologies

* HTML5
* CSS3
* JavaScript (ES6+)

---

## 🎯 Features

* 📋 Display burger menu
* 🔍 Filter items using `filter`
* 💰 Calculate total price with `reduce`
* 🔄 Render items dynamically with `map`
* 📌 Iterate through data with `forEach`

---

## 💡 What I learned

* How to use `map`, `filter`, `reduce`, and `forEach` in real-world scenarios
* How to dynamically update the DOM based on user interactions
* How to structure a simple but functional JavaScript application
