# 🍔 Burger Shop

A dynamic burger ordering interface focused on practicing JavaScript array methods and DOM manipulation in real-world scenarios.

🔗 **Live Demo:** https://jeanholanda.github.io/burger-shop/

---

## About the project

This project is a dynamic interface developed to simulate a burger ordering system using JavaScript.

It allows users to interact with a menu, filter items and calculate totals, focusing on real-world usage of array methods and dynamic rendering.

---

## Sobre o projeto

Este projeto é uma interface dinâmica desenvolvida para simular um sistema de pedidos de hamburguer utilizando JavaScript.

Ele permite que o usuário interaja com um cardápio, filtre itens e calcule valores, com foco na aplicação prática de métodos de array e renderização dinâmica.

---

## 🎯 Purpose

The main goal of this project is to improve my JavaScript skills, especially:

- Working with array methods  
- Manipulating the DOM dynamically  
- Building interactive user interfaces  

---

## 🚀 Technologies

- HTML5  
- CSS3  
- JavaScript (ES6+)  

---

## ✨ Features

- 🍔 Displaying dynamic burger menu  
- 🔍 Filtering items using `filter`  
- 💰 Calculating total price with `reduce`  
- 🔄 Rendering items dynamically with `map`  
- 📌 Iterating through data with `forEach`  

---

## 💡 What I learned

- Using `map`, `filter`, `reduce` and `forEach` in real scenarios  
- Manipulating the DOM based on user interactions  
- Structuring a dynamic JavaScript application  
- Updating UI elements dynamically  
- Improving code organization and readability
