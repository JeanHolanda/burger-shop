# 🍔 Burger Shop

## 🇺🇸 About the project

A simple burger ordering interface built with **HTML, CSS, and JavaScript**, focused on practicing array manipulation methods such as:

* `forEach`
* `map`
* `filter`
* `reduce`

The project simulates a menu where users can view products, filter options, and calculate totals dynamically.

---

## 🇧🇷 Sobre o projeto

Uma interface simples de pedidos de hamburguer, desenvolvida com **HTML, CSS e JavaScript**, com foco na prática de métodos de manipulação de arrays:

* `forEach`
* `map`
* `filter`
* `reduce`

O projeto simula um cardápio onde o usuário pode visualizar produtos, filtrar opções e calcular valores dinamicamente.

---

## 🚀 Technologies

* HTML5
* CSS3
* JavaScript (ES6+)

---

## 🎯 Features

* 📋 Display burger menu
* 🔍 Filter items (e.g., vegetarian, promotions)
* 💰 Calculate total price using `reduce`
* 🔄 Dynamic rendering with `map`
* 📌 Iteration using `forEach`

---

## 💡 What I learned

* How to manipulate arrays in JavaScript
* Practical usage of `map`, `filter`, `reduce`, and `forEach`
* DOM manipulation
* Building dynamic interfaces without frameworks
